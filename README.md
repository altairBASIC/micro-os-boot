# micro-os-boot

Proyecto de boot loader minimalista para un micro sistema operativo.

## Estructura del proyecto

```
micro-os-boot/
├── src/
│   └── boot.asm        # Código fuente del boot loader
├── build/              # Archivos compilados (ignorados por git)
├── docs/
│   ├── informe.md      # Informe del proyecto
│   └── capturas/       # Capturas de pantalla
├── README.md
├── Makefile
└── .gitignore
```

## Compilación

```bash
make
```

## Limpieza

```bash
make clean
```
